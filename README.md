# Визуализация звонка
  1. Симуляция звонка при нажатии на одноименную кнопку 
  2. Кнопки включения и выключения вибраци
  3. Кнопки включения и выключения музыки при звонке (в стадии разработки)
  4. Кнопка тестирования звукового сопровождения звонка (в разработке)

## Окно звонка
  При нажатии на кнопку "Имитировать входящий звонок" появляется окно, содержащее следующее:
  1.  Имя звонящего
  2.  Номер звонящего
  3.  Кнопки "Ответить", "Отклонить", "Создать обращение"
  
  Имя и номер берутся из предзаданного списка в renderer.js (в последствии будет заменён на данные из API базы)
  
  При нажатии на кнопку "Ответить" открывается карточка абонента с автоматическим подставлением имени и номера звонящего
  При "Отклоненить" и "Создать обращение" окно закрывается
